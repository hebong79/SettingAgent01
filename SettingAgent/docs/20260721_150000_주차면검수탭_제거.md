# 주차면 검수 탭 제거

- 날짜: 2026-07-21
- 브랜치: `feat/remove-inspect-tab`

## 변경 내용

`SettingAgent/web/index.html` 상단 탭 네비게이션에서 `주차면 검수` 버튼 1줄을 삭제했다.

```html
<button class="tab" data-tab="inspect">주차면 검수</button>   <!-- 삭제 -->
```

남은 탭: 제어·모니터링 / 정밀 수집 / 분석 / 옵션 / DB.

## 영향도 분석

- `app.js`의 `setTab()`은 `analyze` / `options` / `db` / `precise`만 분기 처리한다. `inspect`는 어떤 분기에도 걸리지 않아 `control`(기본 뷰포트+패널)과 **완전히 동일한 화면**을 보여주는 사실상 무동작 탭이었다. 따라서 제거해도 기능 손실이 없다.
- 전체 소스(`SettingAgent/**`, `.md` 제외)에서 문자열 `inspect` 사용처는 index.html 뿐이었다(그 외에는 헤더 부제 `Camera PTZ · ROI Inspection` 문구만 존재). 삭제로 고아가 되는 JS 함수·CSS 규칙·핸들러 없음.
- ROI 오버레이·주차면 목록 편집 기능 자체는 `제어·모니터링`/`정밀 수집` 탭에서 그대로 동작한다.
- 헤더 부제 문구 `Camera PTZ · ROI Inspection`은 요청 범위 밖이라 **그대로 유지**했다(필요 시 별도 요청).

## 검증

- 정적 검증: `data-tab="inspect"` 잔여 참조 0건, `setTab()` 분기 영향 없음.
- 탭 전환 결선(`document.querySelectorAll('.tab')`)은 남은 버튼들에 그대로 적용되며, 초기 활성 탭은 `control`로 변함없음.
