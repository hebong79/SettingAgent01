# 센터라이징 근본재설계 — lpd 신원 기반 소유권(배타성) 게이트

> 2026-07-20 이터레이션 2. 이터1(pre-aim·저장3중, `20260720_130900_*.md`) 위에 근본해결을 얹음.

## 1. 배경 / 증상
정밀수집 센터라이징에서 "한 주차면 중복 센터라이징 + 다른 주차면 건너뛰기"가 재발.
이터1의 pre-aim(슬롯별 선조준)으로도 해소 안 됨.

## 2. 근본원인 (실제 DB 실증)
`data/setting.sqlite` 를 slot_id 순으로 덤프하면 각 슬롯 lpd 중심은 **뚜렷이 구별**됨(preset2 cx = 0.097 / 0.212 / 0.325 / **0.464** / 0.62 / **0.766**). 순회는 실제로 slot_id 1:1이라 구조상 반복·누락 불가.
그런데 결과 PTZ 는 slot11(lpd cx0.464)·slot13(lpd cx0.766)이 **둘 다 pan≈55.5 로 수렴**(완전히 다른 판인데 중복), slot12 미검.

- **정체:** `PtzCalibrator.calibrateSlot` → `PlatePtz` 는 lpd 를 "목표"가 아니라 "초기 힌트"로만 쓰고, 최종 수렴 대상을 매 프레임 **화면중앙 최근접 재검출(`pickNearestPlate`)** 로 정함. "몇 번 슬롯 처리 중"(순회 1:1)과 "화면 어느 판으로 수렴"(재검출 nearest)이 **분리** → 밀집 프리셋(간격 0.11)에서 줌인 중 이웃 판으로 갈아탐(latch).
- pre-aim 은 시작점만 옮길 뿐 수렴은 여전히 nearest 의존 → 근본 아님.

## 3. 수정 — 소유권(Voronoi) 게이트
센터링 재검출 대상선정을 "화면중앙 최근접" → **"자기 슬롯 lpd 소유권"** 으로 전환. discovery 의 검증된 `pickOwnedPlate`(자기 앵커가 모든 peer 앵커보다 엄격 최근접인 판만 자기 소유) 재사용.

| 파일 | 변경 |
|------|------|
| `src/calibrate/platePtz.ts` | `PlatePtzOpts.peerOffsets?` 추가. `captureAndDetect` 선정 분기(peerOffsets 있으면 `pickOwnedByOffsets`, 없으면 기존 `pickNearestPlate` — 하위호환). `pickOwnedByOffsets` 헬퍼(selfRef=prior 중심 + 상대오프셋 → peerAnchors → `pickOwnedPlate`). 제어 폐루프 수식 무접촉. |
| `src/calibrate/PtzCalibrator.ts` | `run` 프리셋별 `byPreset` 그룹핑. `peerOffsetsFor(t, group)`(자기제외 타슬롯 lpd중심 상대오프셋, near-zero 드롭). `calibrateSlot(t, peerOffsets)` 가 center/zoom **두 인스턴스 모두**에 주입. |

### 좌표계 우회 (핵심 판단)
`pickOwnedPlate` 는 원본 프레임 좌표 규약인데 PlatePtz 는 물리 이동한 현재 프레임에서 검출. 절대 PTZ→원본 역변환(게인 의존, 비-cam1 부정확) 대신 **상대 오프셋 불변성**으로 우회 — 센터링은 zoom 고정이라 pan/tilt=강체평행이동이고 강체평행이동은 Voronoi 소유권을 보존하므로, peer 를 selfRef(prior) 기준 상대오프셋으로 표현하면 게인·절대PTZ 무의존.

### 소유권의 실효 범위 (정직한 한계)
peer 가 selfRef 에 상대고정되므로 **자기 판이 시야에 있는 프레임에선 소유권==최근접**(동일 결과). 소유권의 고유 효과는 **"자기 판 이탈 시 이웃 절도(silent latch) → 정직한 미검(no_plate) 전환"**. 즉 중복수렴 증상을 제거(중복 대신 correct-or-honest-miss)하되, 자기 판을 시야·중앙에 두는 가용성은 pre-aim 몫(상보적).

### near-zero peer 드롭 (PEER_MIN_OFFSET=0.02)
두 슬롯 lpd 중심이 일치/근접하면 peer 앵커가 selfRef 와 겹쳐 엄격부등호 `dSelf<dPeer` 가 항상 false → **모든 검출 기각(두 슬롯 다 미검)**. 겹친 peer 는 Voronoi 판별정보가 없으므로 제외 → 그 쌍은 최근접으로 안전 복귀(단일슬롯 규약과 일관). 실측 판 간격 ≈0.11~0.15 이라 진짜 이웃은 전부 통과.
